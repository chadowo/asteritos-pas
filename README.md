# Asteritos-PAS

This is a port of my previous game jam game Asteritos to the Free Pascal compiler,
using SDL3.

Yes, I know it's probably a worthless game, bad too. But I kinda just wanted to have some
fun doing something ”easy” and familiar, so you”ll have to forgive me on that one.

## Compile

We use Ruby's Rake as build tool here. I didn”t feel like particularly using the Lazarus toolchain to build this. It”s not that complex

### Building

```console
$ rake build
```

The output and artifacts will be in the `build/` directory.
